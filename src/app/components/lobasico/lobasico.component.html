HTML es el lenguaje de marcado estándar para crear páginas web, que actualmente está en su quinta revisión la cuál en conjunto con CSS (el cuál no será abordado en este tutorial), define los nuevos estándares del desarrollo web, rediseñando el código para resolver problemas y actualizándolo así a nuevas necesidades. Con esto no se limita solamente en nuevas etiquetas o atributos, sino que se incorporan muchas caracteristicas nuevas y proporciona una plataforma robusta de desarrollo de complejas aplicaciones web.


  • HTML describe la estructura de una página Web.
  • HTML consta de una serie de elementos los cuales le indican al navegador cómo mostrar el contenido.

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Explicación de la estructura de la página

  • La Línea <!DOCTYPE html> define que el documento es un documento de HTML 5.
  • Los elementos <html> </html> son los elementos raíz de un documento HTML.
  • Los elementos <head> </head> contienen meta información sobre la página HTML.
  • Los elementos <title> </title> son los que especifican el titulo para la página HTML. (El cual se muestra en la barra de titulo del navegador)
  • Los elementos <body> </body> definen el "cuerpo" de la página HTML, es decir son el "contenedor" de todos los elementos visibles de la página tales como: encabezados, párrafos, imagenes, hypervinculos, tablas, listas, etc.
