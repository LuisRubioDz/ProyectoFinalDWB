El atributo style se usa para agregar estilos a los elementos HTML, los cuales pueden ser: color, tipo de fuente, tamaño de fuente, y un muy muy largo etcétera, este atributo tiene la siguiente sintaxis:

<tagname style="property:value;">

La propiedad "Property" es una propiedad de CSS (Cascade Style Sheet - Hojas de estilo en cascada). Lo especificado en "value" también es un valor de CSS.

Nota: Este tutorial no abarcara aspectos avanzados de CSS, ya que es otro tema bastante amplio el cual queda fuera del alcance de este pequeño manual.

Algunos ejemplos de estilos serían los siguientes:

Background color: esta propiedad CSS define el color de fondo ya sea de un elemnto HTML o bien de toda una página, que de hecho es también un elemento HTML, recuerde el elemento: <body> de HTML.


<body>
<h1 style="background-color:blue;">Esto es un encabezado con un color de fondo diferente.</h1>
<p style="background-color:tomato;">Esto es un Párrafo con un color de fondo diferente.</p>
</body>

Esto es un encabezado con un color de fondo diferente.

Esto es un Párrafo con un color de fondo diferente.

Text Color: esta propiedad CSS define el color del texto de cualquier elmento HTML.


<h1 style="color:blue;">Este es un encabezado con el texto color azul.</h1>
<p style="color:red;">Este es un párrafo con el texto color rojo.</p>

Este es un encabezado con el texto color azul.

Este es un párrafo con el texto color rojo.

Text Size: esta propiedad CSS define el tamaño del texto de cualquier elmento HTML.


<h1 style="font-size:300%;">Este es un encabezado con texto GRANDE.</h1>
<p style="font-size:160%;">Este es un párrafo también con texto GRANDE.</p>
