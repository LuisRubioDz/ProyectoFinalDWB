Los links se encuentran en practicamente todas las páginas ya que son los que permiten a los usuarios ir de página en página, o bien a diferentes secciones en una misma página, otra caracteristica es que cuando se pasa el mouse sobre un link el puntero cambia a una pequeña mano.

La equiqueta HTML <a> define un link o hyperlink y su sintaxis es la siguiente:

<a href="url destino">Texto de Link visible al usuario. target="_blank" </a>

El atributo más importante del tag <a> es el atributo <href> que es el que determina el link de destino.

El atributo <target> especifica donde se va a abrir la página referenciada y puede tener alguno de los siguientes valores:

  • _self - Default, abre la página destino en la misma ventana o pestaña donde estemos.
  • _blank - Abre la página destino en una nueva ventana o en una nueva pestaña.
  • _parent - Abre la página destino en el frame parent.
  • _top - Abre la página en la seción body de la ventana donde se dio clic.

Por default todos los links van a aparecer de la siguiente forma en todos los navegadores:

  • Los links no visitados son mostrados subrayados y en color azul.
  • Los links ya visitados son mostrados subrayados y en color púrpura.
  • Los links activos son mostrados subrayados y en color rojo.

Por supuesto que los links pueden ser formateados usando CSS.

HTML Images

Las imágenes pueden mejorar el diseño y la apariencia de una página web. El elemento HTML <img> se usa para incrustar una imagen en una página web, este elemento tiene 2 atributos requeridos:

  • src - Especifica el path (ruta) hacia la imagen.
  • alt - Especifica un texto alterno a la imagen, para los casos en que por cuaqluier razón esta no puede ser cargada.
<img src="Path de la imagen" alt="Texto alterno.">

Siempre defina el tamaño de la imagen a mostrar.

Se puede usar el atributo style para definir la altura y lo ancho de una imagen, ambos valores siempre representan pixeles. Se recomienda siempre definir estos valores para evitar que la página parpadee mientras se carga la imagen.


Imágenes que funcionan como Enlaces.

Para usar una imagen como un enlace basta con poner un elemento <img> dentro de <a>.
