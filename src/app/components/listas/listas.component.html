Las listas HTML permiten agrupar un conjunto items relacionados en listas. Y dichas listas pueden ser ordenadas o no ordenadas.

Listas No Ordenadas

Estas listas inician con el elemento <ul> y cada uno de los items de la lista inicia con el elemento <li>. Cada uno de los items de la lista tendrá por default un bullet (pequeño circulo negro).

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

Listas Ordenadas

Estas listas inician con el elemento <ol> y cada uno de los items de la lista inicia con el elemento <li>. Cada uno de los items de estas listas serán marcados por números de forma automática.

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>
