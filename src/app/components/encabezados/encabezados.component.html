Un elemento HTML casi siempre va a estar definido por un tag de inicio, contenido y un tag de cierre.

<TagHtml>Contenido del elemento</TagHtml>

Algunos ejemplos de elementos HTML se muestran a continuación:

<h1> Ejemplo de encabezado</h1>
<p> Ejemplo de párrafo.</p>

Nunca omitir el tag de cierre de los elementos HTML:

En algunas ocasiones algunos elementos HTML se desplegaran de forma correcta aun y cuando se omita el tag de cierre de dicho elemento, sin embargo no se debe de confiar en que esto siempre sucederá de esta forma, ya que pueden presentarse resultados y errores innesperados cuando se olvida colocar el tag de cierre.

Los elementos HTML no son case sensitive.

Esto quiere decir que NO existe diferencia alguna entre letras mayúsculas y minúscular por lo tanto:

El: <P> significa exactamente lo mismo que <p>.

Hay elementos HTML que no requieren contenido y son llamados elementos vacíos, los cuales tampoco llevan un tag de cierre, a continuación se muestran algunos de estos elementos:

  • El elemento <br> realiza un salto de linea, y es un elemento vacío que no requiere tag de cierre.
  • El elemento <hr> inserta una línea recta, y también es un elemento vacío que no requiere tag de cierre.
