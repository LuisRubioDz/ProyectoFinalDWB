Los encabezados de HTML son títulos o subtítulos que podemos usar para desplegar texto en una página Web. Y se definen con las etiquetas: <h1>, <h2>, <h3>, <h4>, <h5>, <h6>. Donde h1 es el encabezado en el que pondremos lo más importante, y h6 el encabezado con lo menos importante.

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de las páginas web, por lo que es importante usar elementos h1 para los encabezados principales, elementos h2 para poner encabezados menos importantes, h3 y así sucesivamente.

Nota Importante: Los elementos de encabezado de HTML deben de usarse sólo para poner encabezados, es decir no se deben usar estos elementos si lo que desea es hacer el texto más grande o ponerlo en negritas.

Estos sería los elementos HTML para definir encabezados:

<h1>Encabezado más importante.</h1>
<h2>Encabezado menos importante 2.</h2>
<h3>Encabezado menos importante 3.</h3>
<h4>Encabezado menos importante 4.</h4>
<h5>Encabezado menos importante 5.</h5>
<h6>Encabezado menos importante 6.</h6>

y de esta forma es como el navegador los mostraría:

Encabezado más importante.

Encabezado menos importante 2.

Encabezado menos importante 3.

Encabezado menos importante 4.

Encabezado menos importante 5.
Encabezado menos importante 6.

Aunque cada elemento de encabezado de HTML tiene su propio tamaño por default dicho tamaño se puede modificar mediante el atributo style y usando la propiedad font-size de CSS, quedaría como se muestra a continuación:

<h1 style="font-size:60px;">Encabezado más importante.</h1>
